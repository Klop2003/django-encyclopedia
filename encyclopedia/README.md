# Django Encyclopedia

Онлайн-энциклопедия на Django с возможностью выбора светлой и тёмной темы.
Выбранная тема сохраняется в cookies браузера.

## Функционал
- Отображение статей (данные заданы в коде)
- Переключение светлой/тёмной темы
- Сохранение настроек пользователя через cookies
- Использование шаблонов, static-файлов и форм
